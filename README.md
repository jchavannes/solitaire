## Solitaire Golf

App for solving solitaire in as few moves possible.

#### Run
```bash
go run main.go
```
---
#### Download dependencies
```bash
govendor fetch +missing
```
---
#### If needed, install [govendor](https://github.com/kardianos/govendor)
```bash
go get -u github.com/kardianos/govendor
```
